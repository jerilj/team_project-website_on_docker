Setup a web server using docker



Click the following link to download docker and follow the instructions to download and install docker desktop in your local machine. This installation includes Docker Engine, Docker CLI client, Docker Compose etc.

After docker desktop is installed, open the prompt/terminal and enter the command ‘docker -v’. If the docker version is displayed as shown below that means the installation was successful.

Download the ubuntu docker image from docker hub, a public docker registry using the command ‘docker pull ubuntu:latest’. We will be using ubuntu as the operating system to host our web server. As you can see ‘ubuntu’ is the name of the image and ‘latest’ is the tag.

Enter the command ‘docker images’ to see the list of docker images in your local machine

Run the ubuntu container and login to the container using the following command

docker run -it ubuntu:latest /bin/bash

One you login to the container update ubuntu using the command ‘apt-get update’

Install the web server apache2 in the ubuntu container using the following command

apt-get install apache2

Install the command line text editor nano by running the command

apt-get install nano

Exit the container by entering the command ‘exit’

Create a new docker image from the ubuntu image using “docker commit” command

If you enter ‘docker images’ you will be able to see the new image ‘ubuntu_apache2’

Bind a host directory to the container using the following command.

docker run -it -p 127.0.0.1:80:80 -v /Users/jessy/webapp:/var/www/html ubuntu_apache2 /bin/bash

/Users/jessy/webapp is the host directory and /var/www/html is the directory in the container

Create a new html file in the directory /var/www/html using the command ‘nano index.html’.

Write a simple html page

Use CTRL+X and then press ‘Y’ to save the index.html file

Start the apache server by entering ‘/etc/init.d/apache2 restart’

Open a browser and go to the url ‘127.0.0.1’ and you should be able to see the html page you created

Go to the location /User/jessy/webapp in your local machine and you should see the index.html there
