A list of basic Linux commands



pwd - shows current directory

ls - list files and folders in current directory

mkdir - create new directory

cd - change directory

cd .. - change directory

whoami - display current user

mv - mv files

rm - delete files

rm -r - delete directory

docker -v - shows docker version

docker images - shows docker images in local machine

docker run -it ubuntu:latest /bin/bash - login to container

apt-get update - update ubuntu

apt-get install apache2 - install package apache2

exit - exit docker container

docker commit - create new image from an existing image

Bind a host directory to the container using the following command.

docker run -it -p 127.0.0.1:80:80 -v /Users/jessy/webapp:/var/www/html ubuntu_apache2 /bin/bash

/Users/jessy/webapp is the host directory and /var/www/html is the directory in the container
